# IMDB Movies Data Analysis using SQL

## 📌 Project Overview
This project analyzes the IMDB movies dataset using SQL. The goal is to extract meaningful insights related to movies, directors, popularity, revenue, and ratings.

---

## 🧩 Dataset Description

### Tables Used:
- **movies**
- **directors**

### Key Columns:
- Movie details: title, budget, revenue, popularity, ratings
- Director details: name, gender

---

## 📥 Data Source & Retrieval

The dataset used in this project was provided as part of an internship learning program.  
Data was accessed by connecting to a structured relational database using SQL tools.

The data was retrieved by:
- Establishing a connection to a remote database server using a SQL client (e.g., MySQL Workbench)
- Exploring available tables using SQL commands
- Extracting required data using SELECT queries
- Performing analysis directly on the database tables

⚠️ Note: Sensitive connection details such as database credentials, host, and access configurations are not included for security reasons.

## 📊 Key Analysis Performed

### ✔ Data Exploration
- Retrieved full dataset from movies and directors tables

### ✔ Data Cleaning
- Checked for NULL values across all columns

### ✔ Filtering & Conditions
- Found specific directors
- Used pattern matching (LIKE)

### ✔ Aggregations
- Counted total movies
- Counted female directors

### ✔ Ranking & Sorting
- Top 3 popular movies
- Top 3 highest revenue movies

### ✔ Date Filtering
- Highest rated movie after 2000

### ✔ Joins
- Combined movies and directors tables

### ✔ Advanced Analysis
- Director with most movies
- Most bankable director
